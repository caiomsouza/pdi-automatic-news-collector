SETUP INSTRUCTIONS:
1. Download and unpack spark binary distribution
2. Copy your CDH cluster configuration files (core-site.xml, hdfs-site.xml and yarn-site.xml ) to some location
2. Make sure HADOOP_CONF_DIR/YARN_CONF_DIR environment variable is available to spark (set in your env, in spoon or in spark's configuration) and points to the directory you've put cluster configuration files
3. Make sure spark's spark.yarn.jar configuration parameter is pointing to cloudera's spark-assembly.jar (update spark config files or parameters tab of the job entry)*
4. Edit Spark PI job entry's paths to spark-submit utility and application jar to match your environment

* NOTE: By default cloudera manager puts spark-assembly.jar to hdfs://<hdfs file system name>/user/spark/share/lib/spark-assembly.jar during spark installation
