This example produces a multi level XML document from one Excel source.


The sample  is a redesign of the old sample in Kettle's sample folder and makes use 
of the new XML Join Step which allows to add XML structures into an existing XML structure.

The redesign uses 21 steps instead of the 35 of the orirginal sample.
