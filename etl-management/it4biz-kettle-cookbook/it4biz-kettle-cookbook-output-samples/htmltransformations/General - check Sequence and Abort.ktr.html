2009-10-22 Jens Bleuel: created
You can also use the row number fieldname
in the CSV file input step to add the sequence
but this is more flexible.
Check if the sequence numbers are OK , i.e. there are no 'missing' numbers and also no duplicate numbers.
This sample Aborts the complete transformation if there is a difference.
Depending on the commit size within Table output there will be a complete rollback.
Enter Table output properties
and enable this hop to work.
Instead of the Filter rows step you can also
use the Merge rows (diff) step similar.
See sample "Merge rows - mergs 2 streams of data and add a flag"
Enable this hop to abort the transformation.
