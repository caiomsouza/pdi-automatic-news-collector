This sample replaces a number of codes (1 or more) in a field.
Since we don't know how many times we need to replace and we 
want to avoid looping over a transformation, we encode all codes and their replacement full values into a single field.

Drawback: the codes and the full texts can't contain komma (,) or semi-colon (;)

Script 1
//Script here

var name = company_name;

if (name!=null)
{
  var CODE_FULL = CODE_FULL_CODES.split(", ");
  for each (var codeFull in CODE_FULL)
  {
     var pieces = codeFull.split(";");
     var code = pieces[0];
     var full = pieces[1];

     name = replace(name, code, full);
  } 
}
