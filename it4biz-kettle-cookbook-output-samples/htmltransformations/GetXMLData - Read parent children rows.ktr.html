Showing how to read parent/child data from an XML file.
Using the GetXMLData/Read data from XML step.

This step uses XPath to get at the XML data.
Read parent elements from an XML file.

When processing this kind of data for real, you would first process the parent data in 1 transformation
and then the child/detail data in another transformation.

The transformation extracting the parent data can make sure that all parent data exists in the target
before you start processing the child data.
