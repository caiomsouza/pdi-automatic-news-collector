Export Repository

This transformation shows how to to use the JavaScript step to export objects from a repository.

1) Import this transformation into your repository.
2) Run the transformation and provide the following parameters in the run dialog:

DIRECTORY:  The directory in the repository to export.
FILENAME: The file name to save the export as.
