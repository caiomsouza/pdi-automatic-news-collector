Write XML test data 
This example creates performance test data and writes them out to
files/XML Input Stream (StAX) Test 5 - Performance Test Data for Element Blocks.ktr

Note: You need to enable the hop before 'Write XML test data' to get the file created.
See named parameter 'number_of_new_products' for the # of test data.
This parameter defaults to 1 million and produces 1 million test records for the
Analyzer Lists part and 1 million for the Products parts, in total 2 million records.

After creation of the test data, you can replace the file name in the transformation
XML Input Stream (StAX) Test 2 - Element Blocks.ktr
and check the performance.

The file size for the default 1 million will be around 280 MB!
